# 🌊 Rising Waters: Machine Learning-Based Flood Prediction System

A Machine Learning-based web application that predicts flood occurrence using historical weather and rainfall parameters — trained with Scikit-learn and served through a Flask web app.

🔗 **Live App:** [https://risingwaters-1.onrender.com/](https://risingwaters-1.onrender.com/)
